How It Works

When a Go security release is announced, the CVE IDs fixed in that release are extracted from the announcement email and recorded along with the release time.

Every hour, Source Spotter checks vuln.go.dev for any pending CVEs. When a CVE is published to the vulnerability database, the GO ID and publication time are recorded.

The "Lag" column shows how long it took for the vulnerability to be published after the Go release. For unpublished vulnerabilities, it shows how long the vulnerability has been pending.
